### 项目结构说明
	├── lib // 插件源码
	│   ├── components // 组件目录
	│   │   └── loading.vue // 组件文件
	│   └── index.js  // 插件入口文件
	├── index.js // 入口文件
	└── package.json  // 包管理文件
	└── README.md  // 使用说明

### 用法
	npm install kxtx-loading --save
	this.$loading.show()来显示加载 
	this.$loading.hide()来关闭加载

### 包位置
	https://www.npmjs.com/package/kxtx-loading
